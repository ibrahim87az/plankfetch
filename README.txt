#############
#PLANKFETCH!#
#############

DESCRIPTION
-----------
a little fetch that displays your username and hostname, your os, your kernel, uptime, and ram!
unfortunatly, it only works on linux (or even freebsd but i dont know..)

INSTALLATION
------------
while there is already a linux binary, you can compile it by running ./build.sh
WARNING: you need gcc to run ./build.sh. if you dont have gcc, install it or compile it yourself with your compiler.

USAGE
-----
for now, the program doesnt have any flags. just run it by typing ./plankfetch (or whatever you named the binary if you decided to compile it yourself)
